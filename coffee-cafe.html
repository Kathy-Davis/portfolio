Coffee Cafe App

Comprehensive Testing of a Food Ordering Web Application

Project Overview

A coffee and food ordering web application featuring product catalog browsing, online ordering with delivery services, and table reservation functionality.

My Role

QA Tester responsible for validating complete customer workflows from menu exploration to order placement and advance booking systems.

Testing Approach

  • Functional Testing: Verified all core features including product search, add-to-cart, checkout, and payment processing
  • UI Testing: Ensured responsive design across desktop, tablet, and mobile viewports
  • Integration Testing: Validated seamless connection between e-commerce features and restaurant reservation system
  • Regression Testing: Re-tested critical paths after bug fixes

Key Test Cases

TC-001: Browse Menu

Verify all menu items display correctly with images, descriptions, and prices

TC-002: Add to Cart

Confirm items can be added, quantities adjusted, and cart updates in real-time

TC-003: Table Reservation

Validate date/time selection, party size, and special request fields work properly

TC-004: Checkout Flow

Test complete order placement through payment confirmation

Sample Bugs Found

Bug #001: Cart Total Miscalculation

Severity: High

Description: When adding multiple items with modifiers (extra shots, syrups), the cart total did not include modifier costs.

Steps to Reproduce:

  1. Select a coffee drink
  2. Add 2 extra shots
  3. Add to cart
  4. Observe cart total (modifier cost missing)

Expected: Total should include base price + modifier costs

Actual: Only base price displayed

Bug #002: Reservation Time Zone Issue

Severity: Medium

Description: Reservations booked by users in different time zones displayed incorrectly in restaurant dashboard.

Tools Used

  • Jira (bug tracking)
  • TestRail (test case management)
  • Chrome DevTools (debugging)
  • BrowserStack (cross-browser testing)

Outcomes

  • Identified 12 critical bugs before release
  • Created 50+ test cases covering all user workflows
  • Collaborated with developers to verify fixes
  • Contributed to smoother launch with fewer production issues
